This HTML tool take a given JSON text representing a UTC date of when something starts and ends (and optionally a BarHeight and Color parameter), and displays it as a time graph.

This tool is useful for visualizing the events and/or performance of something.

The JSON must be correctly formatted. For Regular mode it must contain property names Start and End, that their values must be a string representing a date (preferably a UTC date, else it assumes local date). See example below.

Notes:
  • This script uses JavaScript's date object, which is based on UNIX time and it ignores leap seconds. If you're recording some event into a JSON, and the start and end happens on the leap second and the second after (assuming a positive leap second), it's possible you have an end date a second before the start date, which causes a slight display issue on the graph.

  • Left/Right or a/d keys: Scroll backward/forward through time. Holding down the left mouse button and moving the mouse also works, as well as rolling the scroll wheel.
  • Up/Down or w/s keys (when “Scale vertical length to fit to graph” is unchecked): Scroll up/down. Used to view when bar heights exceed the display height
  • SHIFT + Up/Down or w/s keys: Zoom. Alternatively hold down CTRL and roll the scroll wheel also affects the rolling (will zoom to where the mouse is at though).
