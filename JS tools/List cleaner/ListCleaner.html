This tool will take a given list in text form, and scans for duplicates and blacklists (line by line). This is useful if you have unwanted items or if there are anomalies at certain lines when copying and pasting.

This tool can also be used to extract URLs that have failed on all process runs should a WBGS process duplicate, a bug that often happens when job queues occur.
